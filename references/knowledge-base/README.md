# Knowledge Base

General reference material for building a best-in-class agent orchestration platform. Articles, architectural patterns, concepts, and ideas collected from across the industry.

## Purpose

This isn't a bookmarks folder. Each entry should capture the core insight in a way that's useful months later without re-reading the source. Link the original, but make the summary self-sufficient.

## Structure

Organize by topic area. Use descriptive filenames: `agent-memory-patterns.md`, `eval-driven-development.md`, `durable-workflow-comparison.md`.

### Topic areas

- **Architecture** — system design patterns, data flow, infrastructure decisions
- **Agent patterns** — orchestration, memory, tool use, multi-agent coordination
- **Eval and observability** — testing AI systems, tracing, quality measurement
- **Workflow engines** — durable execution, state machines, checkpointing
- **Industry trends** — where agent tooling is headed, bets worth making
